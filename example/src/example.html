Outras classes (helpers)

paspatur: cria paddings no elemento

clearfix: o elemento com esta classe leva em consideração onde elementos floats terminam

clearer: o elemento com esta classe puxa o final do elemento pra baixo, mesmo que nele haja floats

transition: adiciona efeito de transição em uma alteração no estilo do elemento

space: usado no elemento <hr> não cria a linha, apenas dá o espaço

Componente: Dialog

Dialog com HTML

<div id="formDialog" class="hidden" title="Exemplo Dialog Simples">
	Conteúdo do dialog
</div>
Dialog.open({
	id: '', //Se não informado, é criado um aleatório
	elem: $('#formDialog'), //Elemento HTML que será o corpo do dialog
	style: {}, //Objeto de itens para o parâmetro style
	open: true, //Se true, abre o form assim que cria
	ephemeral: false, //Se true, destrói o elemento html ao fechar
	changeMonitor: true, //Monitor de alteração de inputs dentro do dialog
	classes: '', //String ou array com classes para o dialog
	title: 'Exemplo de dialog',
	buttons: [
		{
			label: 'Salvar',
			classes: 'bgSuccess R4DialogSaver', //A classe R4DialogSaver faz ctrl+enter ser atalho pro botão
			onClick: function(ev) {
				Warning.show('Botão Salvar pressionado');
			}
		},
		{
			label: 'Fechar',
			classes: 'R4DialogCloser' //Essa classe indica que o botão é de fechar
		}
	]
});
